# ManagerPluginMc
A simple ManagerPlugin that logs the death of Players when they die and what they had on them for the use of SMP servers


Build Guide

1️⃣ Clone the repository
git clone https://github.com/TM-SoftWareDev/ManagerPluginMc.git
cd ManagerPluginMc


2️⃣ Build the plugin

Linux / Mac:

./gradlew build

Windows:

gradlew.bat build
This will compile your plugin and create a .jar file.
Output file will be in:
build/libs/ManagerPlugin-1.1.1.jar
